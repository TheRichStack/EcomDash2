# EcomDash2 Work Orders

This folder holds PM-issued task packs for fresh worker agents.

Use the template in this folder and keep each task narrow.

Recommended rule:

- one slice per work order
- one owner per work order
- one acceptance checklist per work order

Worker agents should not be sent into the repo with only a broad objective.

They should receive:

- a specific objective
- the exact files to read
- the exact files they may edit
- the exact V1 references they may inspect
- acceptance criteria
- verification steps
